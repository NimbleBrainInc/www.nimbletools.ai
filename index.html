Get Started with the CLI

Install once, deploy anywhere

Quick Install

# Install the Nimble CLI
$ npm install -g @nimbletools/ntcli

# Verify installation
$ ntcli --version
ntcli v0.1.0

# Create your account and start deploying
$ ntcli auth signup
